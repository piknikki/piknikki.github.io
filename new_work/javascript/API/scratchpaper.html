Notes about JSONs

  • All property names have to be surrounded by double quotes
  • only simple data expressions -- no function calls, bindings, or computations
  • Comments are not allowed
  • Curly brackets around the whole thing
  • Stringify returns a JSON-encoded string
  • Parse takes a string and converts it to the value it encodes
  • Converting a string to native object is called parsing.
  • Converting native object to string is called stringification.

Notes about client-side APIs

Application Programming Interfaces

  • Built on top of core JavaScript
  • Browser (built into browser) (local to computer)
  • Third party (built on third party location) (need to link to third party)
